# Web2 Gyakorlat - Node.js Projekt

Ez egy Node.js alapú webfejlesztési gyakorlat projekt.

## Telepítés és futtatás

1. Függőségek telepítése:
```bash
npm install
```

2. Alkalmazás indítása:
```bash
npm start
```

3. Fejlesztői módban:
```bash
npm run dev
```

## Projekt struktúra

```
├── index.js          # Fő alkalmazás fájl
├── package.json      # NPM konfiguráció
├── src/              # Forráskód
│   ├── routes/       # Útvonal kezelők
│   ├── models/       # Adatmodellek
│   └── utils/        # Segédprogramok
└── public/           # Statikus fájlok (CSS, JS, képek)
```

## Hasznos parancsok

- `npm start` - Alkalmazás indítása
- `npm run dev` - Fejlesztői mód (automatikus újraindítás)
- `npm test` - Tesztek futtatása

## Technológiák

- Node.js
- HTTP szerver
- További csomagok szükség szerint
